# CONCEPTS TO APPLY AND LEARN WITH

- Partial Application
- Currying
- Folding (Linear Fold)
- Unfolding (Linear Unfold)
- Functional Map and Filter
- Lazy Evaluation
- Memoization.

## Setup Instructions

After cloning the repository, run the following commands:

```bash
# Install dependencies
npm install

# Build the project
npm run build

# Run tests
npm test

# Run linting
npm run lint
```
